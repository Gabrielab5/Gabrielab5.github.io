The Popcorn Palace - Movie Booking System

The Popcorn Palace project is a Spring Boot-based web application designed to manage a movie booking system. It supports core features such as creating and retrieving movies, scheduling and validating showtimes without overlaps, and booking seats for showtimes while ensuring no duplicate reservations occur. The system is organized into clear layers: entities (like Movie, Showtime, and Booking), DTOs for request/response handling, controllers to manage RESTful API endpoints, and services for business logic. The application ensures robust data handling with input validation, exception management, and database integration via JPA repositories
