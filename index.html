EnerGenius- Energy Optimization Web Application

The EnerGenius project is a smart energy management platform designed to help users monitor, analyze, and optimize their electricity consumption. It enables users to upload consumption data—such as Excel files from their electricity provider—and receive personalized insights, forecasts, and energy-saving recommendations. The app includes features like consumption trend analysis, alert notifications for high usage, goal tracking, and actionable suggestions tailored to user behavior. With a user-friendly interface and a focus on data privacy, EnerGenius empowers households to take control of their energy use and reduce costs effectively.

The Popcorn Palace - Movie Booking System

The Popcorn Palace project is a Spring Boot-based web application designed to manage a movie booking system. It supports core features such as creating and retrieving movies, scheduling and validating showtimes without overlaps, and booking seats for showtimes while ensuring no duplicate reservations occur. The system is organized into clear layers: entities (like Movie, Showtime, and Booking), DTOs for request/response handling, controllers to manage RESTful API endpoints, and services for business logic. The application ensures robust data handling with input validation, exception management, and database integration via JPA repositories

MedHub- Social Media Web Application

MedHub is a full-stack web application designed to serve as a social media platform for medical staff and professionals. The Frontend is built using React, offering an intuitive and user-friendly interface for users to interact, share updates, and connect with peers in the medical community. The Backend is developed with Node.js, responsible for handling API endpoints, business logic, and database operations to ensure secure and efficient data management. Together, these components deliver a seamless experience tailored for healthcare professionals to collaborate and communicate effectively.
